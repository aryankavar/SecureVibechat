THE ERROR:
  WebRTC init error: TypeError: f.toJSON is not a function

WHAT CAUSED IT:
  The code was calling .toJSON() on WebRTC objects (offer, answer,
  ICE candidates), but this method doesn't exist in all browsers.

WHAT I FIXED:
  ✅ Manually extract properties instead of using .toJSON()
  ✅ Fixed offer signaling
  ✅ Fixed answer signaling
  ✅ Fixed ICE candidate signaling
  ✅ Updated src/utils/webrtcManager.js
  ✅ Created WEBRTC_TOJSON_FIX.md guide
  ✅ Updated TROUBLESHOOTING.md

═══════════════════════════════════════════════════════════════════

THE FIX:

BEFORE (Broken):
  const offer = await peerConnection.createOffer()
  await this.sendSignal(chatId, userId, {
    type: 'offer',
    offer: offer.toJSON()  // ❌ Not available in all browsers
  })

AFTER (Fixed):
  const offer = await peerConnection.createOffer()
  await peerConnection.setLocalDescription(offer)
  await this.sendSignal(chatId, userId, {
    type: 'offer',
    offer: {
      type: offer.type,
      sdp: offer.sdp
    }
  })
